# CONSULTA-CPF-NA-RECEITA-FEDERAL
CONSULTA CPF NA RECEITA FEDERAL
Consulta o CPF na Receita Federal buscando o status do CPF

Sobre este fluxo de automação:
Esse Workflow permite que com os dados de data de nascimento e o número de CPF você possa consultar a situação do CPF junto a receita federal.
Muito útil para analise de cadastros e validação de perfil.
Para usar esse processo é necessário a compra de créditos em um site que faz o bypass do H-Captcha, pode ser o DeathByCatpcha ou outro de sua preferência.
